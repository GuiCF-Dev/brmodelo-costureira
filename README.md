# 🧵 Sistema de Oficina de Costura - Modelagem de Banco de Dados

## 📖 Sobre o Projeto

Este projeto apresenta a modelagem de um **Sistema de Oficina de Costura**, desenvolvida utilizando a ferramenta **brModelo** com o objetivo de representar e organizar as informações necessárias para o gerenciamento das atividades de uma oficina de costura.

O modelo contempla os principais processos do negócio, permitindo o controle de clientes, encomendas, serviços, funcionários, materiais e acompanhamento da produção, proporcionando uma visão estruturada do funcionamento do sistema.

---

## 🎯 Objetivos

* Compreender os conceitos de modelagem de banco de dados.
* Representar processos reais através de diagramas.
* Identificar entidades, atributos e relacionamentos.
* Aplicar regras de normalização.
* Estruturar informações de forma organizada.
* Preparar a modelagem para implementação em um banco de dados relacional.

## 🏭 Contexto do Projeto

A modelagem foi desenvolvida para representar os processos de uma oficina de costura, contemplando informações relacionadas a:

* Clientes
* Encomendas
* Funcionários
* Serviços de Costura
* Materiais e Tecidos
* Produção
* Entregas
* Pagamentos

O objetivo é garantir uma organização eficiente dos dados e facilitar o gerenciamento das atividades da oficina.

## 📈 Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conceitos fundamentais de análise e modelagem de dados, permitindo compreender como sistemas empresariais podem organizar informações de forma estruturada.

A atividade contribui para o entendimento da etapa de planejamento de bancos de dados antes da implementação em sistemas gerenciadores como MySQL, PostgreSQL ou SQL Server.
